# 冬季旅行装备清单

## 必备装备
- [ ] 羽绒服或厚棉服
- [ ] 保暖内衣
- [ ] 厚围巾、帽子、手套
- [ ] 防滑防水靴
- [ ] 暖宝宝
- [ ] 保温杯
- [ ] 保湿霜和润唇膏
- [ ] 墨镜（雪地反光）
- [ ] 充电宝（低温下电池消耗快）

## 滑雪旅行额外装备
- [ ] 滑雪服和滑雪裤
- [ ] 护目镜
- [ ] 防水手套
- [ ] 护具（护膝、护臀）
- [ ] 速干衣

## 注意事项
- 室内外温差大，注意增减衣物
- 冰雪路面行走小心滑倒
- 相机/手机注意保暖，避免自动关机
- 皮肤干燥注意保湿
- 自驾需准备防滑链
