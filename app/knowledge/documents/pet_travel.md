# 带宠物旅行装备与注意事项

## 必备装备
- [ ] 宠物航空箱或车载笼
- [ ] 牵引绳和胸背带
- [ ] 宠物身份证和疫苗证明
- [ ] 宠物粮和零食
- [ ] 便携水碗和食盆
- [ ] 宠物湿巾和拾便袋
- [ ] 宠物常用药品
- [ ] 宠物毯子或垫子
- [ ] 宠物玩具

## 注意事项
- 提前确认酒店/民宿是否允许宠物入住
- 查询目的地对宠物的相关规定
- 长途车程每隔 2-3 小时休息一次
- 不要让宠物单独留在密闭车内
- 携带宠物近期照片，以防走失
- 准备宠物急救包
- 提前找好目的地附近的宠物医院

## 交通建议
- 自驾最方便，可自由安排休息
- 高铁/飞机需提前办理宠物托运
- 了解航空公司宠物托运要求
- 准备宠物镇定药物（遵医嘱）
