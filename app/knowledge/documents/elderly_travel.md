# 带老人旅行注意事项

## 必备装备
- [ ] 常用药品（降压药、降糖药、心脏病药等）
- [ ] 血压计/血糖仪
- [ ] 舒适的平底鞋
- [ ] 保温杯
- [ ] 轻便折叠凳
- [ ] 老人身份证和医保卡
- [ ] 紧急联系人卡片
- [ ] 保暖衣物

## 注意事项
- 行程节奏要慢，避免早出晚归
- 每天步行时间不超过 2-3 小时
- 选择无障碍设施完善的景点和酒店
- 餐厅选择清淡、卫生的
- 随身携带急救药品
- 避免高原、高寒、高温等极端环境
- 购买含紧急医疗的旅游保险

## 交通建议
- 优先高铁一等座或飞机商务舱
- 避免红眼航班
- 机场/车站可申请轮椅服务
