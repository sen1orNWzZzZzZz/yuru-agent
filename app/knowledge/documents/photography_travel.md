# 摄影旅行装备清单

## 必备装备
- [ ] 相机和镜头
- [ ] 三脚架
- [ ] 备用电池（多带几块）
- [ ] 存储卡（多带几张）
- [ ] 读卡器
- [ ] 相机清洁套装
- [ ] 防水相机包
- [ ] 便携反光板
- [ ] 快门线或遥控器

## 注意事项
- 日出日落提前到达拍摄点
- 注意保护器材，防尘防水
- 备份照片到云端或移动硬盘
- 了解目的地是否允许无人机飞行
- 尊重当地拍摄禁忌
- 携带备用存储卡和电池

## 特殊场景
- 星空摄影：大光圈广角镜头、头灯、保暖衣物
- 人像摄影：定焦镜头、反光板、补光灯
- 野生动物摄影：长焦镜头、 camouflage 服装
