## 演示流程（3个页面，6个核心功能点）

### 页面1：首页（需求输入）
**URL：** `/index.html`

**功能展示：**
1. **灵动光球动画** - 页面中央的发光球体带有呼吸动画和脉冲光晕
2. **自然语言输入** - 预填充演示文案："下个月带爸妈去杭州，想看西湖和灵隐寺，不要太累，预算人均3000"
3. **快捷参数调节** - 出发日期/结束日期/出行人数（已预设3人）
4. **快捷标签** - 家庭出游+慢节奏已选中
5. **一键规划** - 点击"开始规划"按钮

**操作步骤：**
```
1. 打开首页
2. 观察灵动光球的呼吸动画和浮动粒子背景
3. 查看已预填充的演示需求
4. 点击"开始规划"按钮
5. 系统自动加载预生成的Agent编排结果，跳转看板页
```

### 页面2：Agent协作看板
**URL：** `/planning.html`

**功能展示（6个Agent卡片）：**

#### Agent 1: 中央调度器
- 展示解析出的关键词标签（杭州、爸妈、西湖、灵隐寺等）
- 显示提取的结构化信息：目的地/人数/风格/预算

#### Agent 2: 酒店推荐Agent
- 展示3家推荐酒店（杭州西湖国宾馆、千岛湖洲际、法云安缦）
- **关键特性：** 每个酒店显示口碑分数（如85分）

#### Agent 3: 餐饮推荐Agent
- 展示3家推荐餐厅（楼外楼、知味观、桂语山房）
- **关键特性：** 显示距离信息（如1.2km），反水军降权提示

#### Agent 4: 景点推荐Agent
- 展示3个推荐景点（西湖风景区、灵隐寺、千岛湖）
- **关键特性：** 高反风险评估警告（海拔相关）

#### Agent 5: 交通规划Agent
- 展示3种交通方案（高铁二等座/一等座/自驾）
- **关键特性：** 耗时和价格对比

#### Agent 6: 风控专家Agent
- **关键特性：** 发现2个异常并给出建议
- 展示风险等级（medium/high）和具体建议

**操作步骤：**
```
1. 观察6个Agent卡片以动画方式依次入场
2. 展开/折叠任意Agent卡片查看详情
3. 注意餐厅Agent卡片底部的水军嫌疑警告
4. 点击"查看完整行程"按钮
```

## 本地运行完整Python后端

### 环境要求
- Python 3.9+
- pip

### 安装步骤

```bash
# 1. 进入项目目录
cd python-travel-agent

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
python run.py

# 4. 浏览器访问
# http://127.0.0.1:8000
```

### 验证后端API

```bash
# 健康检查
curl http://127.0.0.1:8000/api/ping

# Agent编排（核心API）
curl -X POST http://127.0.0.1:8000/api/agent/orchestrate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "下个月带爸妈去杭州，想看西湖和灵隐寺，不要太累"}'

# 搜索酒店
curl "http://127.0.0.1:8000/api/recommendation/hotels?city=杭州&max_price=2000"

# 小红书POI口碑
curl "http://127.0.0.1:8000/api/xiaohongshu/poi?poi_name=西湖风景区"

# 作者信誉
curl "http://127.0.0.1:8000/api/xiaohongshu/author?author_id=author_spam_001"
```
